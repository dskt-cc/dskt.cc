title: Publishing Standards
description: To list your mod on dskt.cc, include a `dskt.json` file in the root of your repository.
---

## Schema Structure

```json
{
  "author": "YourGitHubUsername",
  "version": "1.0.0",
  "type": "MelonLoader",
  "category": ["Performance"],
  "description": "A brief description of your mod",
  "requires": ["ModName1", "ModName2"],
  "locale": {
    "default": "en-US",
    "supported": ["en-US", "ja-JP"]
  },
  "sponsor": {
    "name": "Sponsor Name",
    "url": "https://sponsor.url"
  }
}
```

## Required Fields

- **author**: Your GitHub username (alphanumeric and hyphens only)
- **version**: Current version (semver format)
- **type**: Mod loader type:
  - `"MelonLoader"`: MelonLoader mod
  - `"BepInEx"`: BepInEx plugin
  - `"Both"`: Supports both loaders
- **category**: Array of categories from:
  - `"Performance"`: Performance improvements
  - `"QualityOfLife"`: Convenience features
  - `"Content"`: New features or content
  - `"Overhaul"`: Major changes
  - `"Other"`: Misc modifications
- **description**: Brief description of your mod (10-500 characters)

## Optional Fields

### Dependencies
```json
"requires": ["MateChat", "CustomAvatarLoader"]
```
- Array of mod names that your mod depends on
- Names must match existing mods in the official mod list

### Localization
```json
"locale": {
  "default": "en-US",
  "supported": ["en-US", "ja-JP"]
}
```
- **default**: Default language code
- **supported**: Array of supported language codes

### Sponsorship
```json
"sponsor": {
  "name": "Sponsor Name",
  "url": "https://sponsor.url"
}
```
- **name**: Sponsor's name
- **url**: Sponsor's website
