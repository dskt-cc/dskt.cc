## Schema Structure

```json
{
  "name": "YourModName",
  "meta": {
    "description": "A brief description of your mod",
    "author": "YourGitHubUsername",
    "category": ["Performance"],
    "type": "MelonLoader",
    "version": "1.0.0"
  },
  "locale": {
    "default": "en-US",
    "supported": ["en-US", "ja-JP"]
  },
  "sponsor": {
    "name": "Sponsor Name",
    "url": "https://sponsor.url"
  }
}
```

## Required Fields

### Core Properties

- **name**: Your mod's name (no spaces)
- **meta**: Object containing mod metadata

### Meta Properties

- **description**: Brief description of your mod
- **author**: Your GitHub username
- **category**: Array of categories from:
  - `"Performance"`: Performance improvements
  - `"QualityOfLife"`: Convenience features
  - `"Content"`: New features or content
  - `"Overhaul"`: Major changes
  - `"Other"`: Misc modifications
- **type**: One of:
  - `"MelonLoader"`: MelonLoader mod
  - `"BepInEx"`: BepInEx plugin
  - `"Both"`: Supports both loaders
- **version**: Current version (semver format)

## Optional Fields

### Localization

```json
"locale": {
  "default": "en-US",
  "supported": ["en-US", "ja-JP"]
}
```

- **default**: Default language code
- **supported**: Array of supported language codes

### Sponsorship

```json
"sponsor": {
  "name": "Sponsor Name",
  "url": "https://sponsor.url"
}
```

- **name**: Sponsor's name
- **url**: Sponsor's website
