## Desktop Mate Modding Community erver
This Discord server is a place where developers and users can coexist in the same place, developers can get feedback on their mods and users can help each other when they encounter issues and have direct contact with developers for bug reports.

- Ask for help if you have issues with any mods shown above
- Get help from other mod developers when developing mods
- Share your non Desktop Mate projects with other developers
- Access to a big VRM archive for Custom Avatar Loader Mod
