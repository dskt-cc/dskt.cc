## Installing Desktop Mate

1. Download Desktop Mate from [Steam](https://store.steampowered.com/app/3301060/Desktop_Mate/)
2. Install the game to your preferred location
3. Run the game once to create necessary folders

## Installing .NET 6 Runtime

1. Visit the [Microsoft .NET download page](https://dotnet.microsoft.com/download/dotnet/6.0)
2. Download the .NET 6 Desktop Runtime
3. Run the installer and follow the prompts

## Installing ModLoaders

### MelonLoader

1. Download MelonLoader from [releases](https://github.com/LavaGang/MelonLoader/releases)
2. Run the installer
3. Select your Desktop Mate installation folder
4. Click Install

### BepInEx

1. Download BepInEx from [releases](https://github.com/BepInEx/BepInEx/releases)
2. Extract to your Desktop Mate installation folder
3. Run the game once to generate config files

## Installing Mods

1. Download mods from [dskt.cc](https://dskt.cc/mods)
2. Place the mod files in the appropriate folder:
   - MelonLoader: `Mods` folder
   - BepInEx: `BepInEx/plugins` folder
3. Launch the game and enjoy your mods!
